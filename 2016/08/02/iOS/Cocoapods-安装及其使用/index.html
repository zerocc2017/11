六. 移除项目中的 Cocoapods

  1. 删除工程文件夹下的Podfile、Podfile.lock及Pods文件夹;
  2. 删除xcworkspace文件;
  3. 使用xcodeproj文件打开工程，删除 Frameworks 组下的 Pods.xcconfig 及 libPods.a引用;
  4. 在工程设置中的 Build Phases下删除 Check Pods Manifest.lock 及 Copy Pods Resources;

如果将 cocoapods 集成到工程中后不小心修改或删除了其相关文件导致无法编译通过例如：不小心把 Pods.xcconfig 给删除了然后出现:

diff: /../Podfile.lock: No such file or directory，

用上面的方法删除cocoapods后，再重新 $sudo pod install 一下就好了。

如果编译的时候出现权限问题，对工程文件夹

$sudo chmod 777 path-to-project-folder/*
$sudo chown 777 path-to-project-folder/*
